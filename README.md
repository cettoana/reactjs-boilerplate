# React.js boilerplate

## Packages

* React, ReactDOM
* webpack 2
* ESlint
* stylelint

## webpack config

* Babel presets
  * ES5, ES6, ES7
  * react
* style-loader
* css-loader
* postcss-loader
  * autoprefixer

## Getting started

### Dev

```
npm install
npm run watch
```

### Lint check

```
npm run eslint
npm run stylelint
```
